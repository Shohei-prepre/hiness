# ハイネス｜営業フォーム自動化システム 説明書

---

## 全体の流れ

```
STEP1 リサーチ       → data/raw_companies.csv        （会社名＋公式URL）
STEP2 フォーム発見   → data/enriched_companies.csv   （フォームURL追加）
STEP3 試運転        → data/safe.csv                  （送信OKな会社だけ）
STEP4 本番送信      → data/sent_log.csv              （送信ログ）
```

---


## STEP1｜リサーチ（会社リスト収集）

**やること：** 求人広告代理店の「会社名＋公式URL」を収集する

**ソース：**
- JESRA（日本人材紹介事業協会）会員322社
- ARNA（全国求人情報協会）リンク集
- PRtimes 人材紹介キーワード

**実行：**
```powershell
python agent1_research.py
```
または `run_agent1.bat` をダブルクリック

**所要時間：** 約6〜8分（JESRAのプロフィールページを1社ずつ訪問するため）

**アウトプット：** `data/raw_companies.csv`

| 列 | 内容 |
|---|---|
| company_name | 会社名 |
| official_url | 公式URL |
| source | 取得元（JESRA / ARNA / PRtimes） |

---

## STEP2｜フォーム発見（問い合わせフォームURL取得）

**やること：** 公式URLから問い合わせフォームのURLを探し出す

**ロジック：**
1. トップページの「お問い合わせ」リンクを優先して探す
2. なければ `/contact/` `/inquiry/` `/お問い合わせ/` など9パターンを総当たり
3. `<form>` + `<textarea>` または `<input>` があればフォームと判定
4. 安全チェック（ドメイン一致・会社名検出・除外パス確認）

**実行：**
```powershell
python agent2_enrich.py
```

**所要時間：** 約30〜40分（267社分）

**アウトプット：** `data/enriched_companies.csv`

| 列 | 内容 |
|---|---|
| company_name | 会社名 |
| official_url | 公式URL |
| form_url | フォームURL |
| status | found / not_found / skip |
| source | 取得元 |

> `status=found` の会社だけがSTEP3に進む（目安：100〜200社）

---

## STEP3｜試運転（Playwright dry-run）

**やること：** Playwrightでフォームを自動入力し、**送信ボタン手前で止まる**

**判定：**
- 必須項目がすべて埋まる → OK
- 入力バリデーションエラーが出ない → OK
- 両方クリアした会社のみ `safe.csv` に記録

**対応フォーム種別：**
- Contact Form 7（WordPress）
- WPForms
- MW WP Form
- 汎用フォーム（上記以外）

**除外（送信しない）：**
- toC企業（個人向けサービス）
- 公的機関・大企業
- 採用・サポートページ
- 第三者プラットフォーム（form-mailer.jp等）

**実行：**
```powershell
python agent3_dryrun.py
```

**アウトプット：** `data/safe.csv`（dry-run通過した会社のみ）

---

## STEP4｜本番送信

**やること：** `safe.csv` の会社に対して実際にフォームを送信する

> ⚠️ **送信前に必ず `config.py` の自社情報・メッセージ本文を設定すること**

**実行：**
```powershell
python agent4_submit.py
```

**アウトプット：** `data/sent_log.csv`

| 列 | 内容 |
|---|---|
| company_name | 会社名 |
| form_url | 送信したフォームURL |
| result | success / failed |
| timestamp | 送信日時 |
| error | 失敗時のエラー内容 |

---

## config.py の設定（STEP4前に必須）

`config.py` を開いて以下を記入する：

```python
SENDER_INFO = {
    "company":   "（自社名）",
    "name":      "（担当者名）",
    "name_kana": "（ふりがな）",
    "email":     "（メールアドレス）",
    "phone":     "（電話番号）",
    "title":     "（役職）",
    "url":       "（自社URL）",
}

MESSAGE_TEMPLATE = """
（営業メッセージ本文をここに記入）
"""
```

---

## Excelに出力する

各STEPのCSVをExcel形式に変換できる：

```powershell
# STEP1完了後
python export_excel.py --input data/raw_companies.csv --output data/ハイネス_リスト_STEP1.xlsx

# STEP2完了後
python export_excel.py --input data/enriched_companies.csv --output data/ハイネス_リスト_STEP2.xlsx

# STEP3完了後（最終送信リスト）
python export_excel.py --input data/safe.csv --output data/ハイネス_送信リスト.xlsx
```
